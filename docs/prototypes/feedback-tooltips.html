Tooltips & Popovers - Feedback Prototype

Interactive tooltip and popover components for the Discord Bot Admin UI

1. Tooltip Positions

Hover over each button to see tooltips in different positions. Tooltips appear after a 200ms delay.

2. Tooltip with Custom Delay

Tooltips support configurable delays. The default is 200ms, but you can customize it.

3. Popover Positions

Click each button to open popovers in different positions. Click outside or press Escape to close.

Top Popover
This popover appears above the trigger element. It supports rich content including text, links, and buttons.
Right Popover
This popover appears to the right of the trigger element.
Bottom Popover
This popover appears below the trigger element.
Left Popover
This popover appears to the left of the trigger element.

4. Popover with Rich Content

Popovers can contain rich HTML content including titles, descriptions, and action buttons.
