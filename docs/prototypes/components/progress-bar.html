Accessibility

ARIA attributes for screen readers and assistive technologies.

Determinate Progress with ARIA

45%

role="progressbar" aria-valuenow="45" aria-valuemin="0" aria-valuemax="100"

Indeterminate Progress with ARIA

role="progressbar" aria-busy="true" (no aria-valuenow for indeterminate)

ARIA Best Practices:

  • Use role="progressbar" for semantic meaning
  • Set aria-valuenow for determinate progress (0-100)
  • Omit aria-valuenow for indeterminate (use aria-busy="true")
  • Always include descriptive aria-label
  • Update aria-valuenow dynamically as progress changes
