Usage Notes

Overlay Z-Index

Full-screen overlays use z-index: 1200 to ensure they appear above all other content including modals and dropdowns.

Backdrop Color

The overlay backdrop uses rgba(29, 32, 34, 0.8) - a semi-transparent dark color matching the design system.

Accessibility

Loading overlays include role="alert" and aria-busy="true" for screen reader support.

Animation Performance

All animations use CSS transforms and opacity for smooth 60fps performance.
