Implementation Notes

  • Embed Color: Use the guild's icon dominant color if extractable, otherwise default to Discord Blurple (#5865F2).
  • Member Counts: Use number formatting with commas for readability (e.g., "1,234,567"). Show breakdown of online vs total.
  • Channel Counts: Group by type (text, voice, categories). Consider including stage channels and forum channels if present.
  • Boost Level: Show current level (0-3) and number of active boosts. Use pink color (#F47FFF) for boost-related info.
  • Features/Badges: Show relevant server features as badges. Common ones: Verified, Partner, Community, Discovery.
  • Vanity URL: Only show if the server has a custom vanity URL configured. Format as clickable link.
  • Description: Show server description if set. Omit the field entirely if no description exists.
  • Thumbnail: Always show the server icon as the embed thumbnail (right side, 80x80px with rounded corners).
  • Owner Mention: Show as @username format. Consider showing display name if different.
