Implementation Notes

  • Embed Color: Use the role's actual color as the embed's left strip. If no color, use Discord's default gray (#99AAB5).
  • Role Name Title: Display role name in the role's color. Show color swatch before name for clarity.
  • Color Display: Show hex code and common color name (e.g., "#3498DB (Blue)"). For no-color roles, show "No color (default)".
  • Dangerous Permissions: Highlight with warning icons (yellow). Include: Administrator, Manage Server, Manage Roles, Manage Channels, Manage Webhooks, Ban Members, Kick Members, Mention @everyone.
  • Managed Roles: Show integration source (bot name, Twitch, YouTube, etc.). Indicate that role cannot be manually assigned.
  • Hoisted: Explain that "displays separately" means the role shows as a separate group in the member list.
  • Member Count: Show number of members with this role. For bot roles, clarify "(bot only)".
  • Position: Show as "#N in hierarchy" where lower numbers = higher position. Consider the @everyone role as the baseline.
  • Booster Role: Special styling with boost icon and pink theme color.

Dangerous Permissions Reference

  • Administrator - Full server control, bypasses all permissions
  • Manage Server - Can modify server settings, invites, vanity URL
  • Manage Roles - Can create/edit/delete roles below their highest role
  • Manage Channels - Can create/edit/delete channels
  • Manage Webhooks - Can create webhooks that post as the server
  • Ban Members - Can permanently ban users
  • Kick Members - Can remove users from server
  • Mention @everyone - Can ping all server members
