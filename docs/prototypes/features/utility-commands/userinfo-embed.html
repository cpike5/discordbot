Implementation Notes

  • Embed Color: Use Discord Blurple (#5865F2) by default. Switch to Booster Pink (#F47FFF) for server boosters.
  • Date Formatting: Show absolute date (e.g., "Dec 31, 2020") with relative time below (e.g., "4 years ago").
  • Role Truncation: Show up to 5-6 roles inline, then display "(+N more)" indicator to prevent embed overflow.
  • Permissions: Only show "key" or dangerous permissions. Common permissions like "Send Messages" can be omitted unless the user has minimal permissions.
  • Bot Badge: Display the BOT badge after the username for bot accounts.
  • Boost Status: Only show the boost field if the user is actively boosting. Omit for non-boosters.
  • Thumbnail: Always show the user's avatar as the embed thumbnail (right side, 80x80px).
  • Nickname vs Display Name: Show "Display Name" if different from username. For bots, omit this field.
