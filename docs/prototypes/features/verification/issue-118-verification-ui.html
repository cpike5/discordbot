Prototype Documentation

Section 1: Initial State

Shows the "Link via Discord Bot" option with a "Start Verification" button. This appears after the OAuth button when the user has no pending verification.

Section 2: Pending Verification State

Shows the verification code entry form with instructions, countdown timer, and input field. The code input is styled with monospace font, uppercase, and centered text for easy code entry.

Section 3: Error State

Shows an error message when verification expires or fails, with a "Try Again" button to restart the process.

CSS Classes Used

  • bg-info-bg, border-info-border, text-info - Info boxes
  • btn btn-primary - Primary action buttons
  • btn btn-secondary - Secondary/outline buttons
  • btn btn-ghost - Tertiary/cancel buttons
  • form-input - Standard input styling
  • verification-code-input - Monospace, uppercase, centered code input
