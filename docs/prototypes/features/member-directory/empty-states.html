Empty State Components

These empty states are displayed when there is no data to show, when filters return no results, or when an error occurs while loading member data.

1. No Members in Server (New Guild)

Displayed when the server has no members or member data has not been synced yet.

No members yet

This server has no member data. Click the button below to sync members from Discord.

2. No Results from Filters

Displayed when the applied filters do not match any members.

No members found

Try adjusting your search or filters to find members.

3. Error Loading Members

Displayed when there is an error fetching member data from the server.

Failed to load members

An error occurred while fetching member data. Please try again.

4. Rate Limit Error (Alert)

Displayed as an alert when the Discord API rate limit is reached.

Rate Limited

Discord API rate limit reached. Please wait a moment and try again. The limit will reset in approximately 30 seconds.

5. Modal - Member Not Found

Displayed inside the member detail modal when the member cannot be found (left server or was removed).

Member Details

Member not found

This member may have left the server or been removed.

6. No Roles Available (Filter Dropdown)

Displayed inside the role filter dropdown when the server has no roles configured.

No roles available

7. Compact Empty States (Inline)

Smaller empty state variants for use in constrained spaces like table cells or sidebars.

Compact - No Data

No members

Compact - No Results

No matches

Failed to load
