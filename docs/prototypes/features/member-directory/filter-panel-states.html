Filter Panel Variations

The filter panel supports multiple states including collapsed/expanded, active filters indicator, role multi-select dropdown, and applied filter badges.

1. Collapsed State (No Active Filters)

Default collapsed state with chevron pointing right. No active filter badge shown.

2. Collapsed State (With Active Filters Badge)

Collapsed with orange badge indicating number of active filters.

3. Expanded State (Full Filter Form)

Fully expanded filter panel with all filter options visible. Chevron points down.

4. Role Multi-Select Dropdown (Open)

Role filter dropdown showing checkboxes with role colors. Multiple roles can be selected.

5. Date Input Focused State

Date input fields with focus ring styling.

6. Applied Filters Badges

When filters are applied, they can be shown as removable badges below the filter panel for quick reference.
