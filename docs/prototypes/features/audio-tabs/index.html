Implementation Notes

Component Location

Pages/Shared/Components/_AudioTabs.cshtml

ViewModel

ViewModels/Components/AudioTabsViewModel.cs

  • - GuildId: Required for building tab URLs
  • - ActiveTab: "soundboard" or "settings"
  • - SoundCount: Optional badge showing number of sounds

Usage

<partial name="Shared/Components/_AudioTabs" model="new AudioTabsViewModel {
    GuildId = Model.GuildId,
    ActiveTab = "soundboard",
    SoundCount = Model.SoundCount
}" />

Accessibility

  • - Uses role="tablist" and role="tab"
  • - Proper aria-selected state on active tab
  • - Descriptive aria-label on navigation
