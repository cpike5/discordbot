Custom Scrollbar Prototype - Issue #801

This prototype demonstrates custom scrollbar styling for the sidebar navigation. The sidebar on the left uses the .custom-scrollbar class to apply themed scrollbar styles that match the dark Discord-inspired design system.

To test: Resize your browser window to make the sidebar content overflow, or scroll through the additional items at the bottom of the sidebar.

Colors Used

The scrollbar uses these design system tokens:

--color-bg-hover (#363a3e) - Thumb
--color-text-tertiary (#7a7876) - Thumb hover
transparent - Track

CSS Implementation

Copy this CSS to apply custom scrollbar styling:

/* Webkit browsers (Chrome, Safari, Edge) */ #sidebar::-webkit-scrollbar { width: 6px; } #sidebar::-webkit-scrollbar-track { background: transparent; } #sidebar::-webkit-scrollbar-thumb { background: var(--color-bg-hover); border-radius: 3px; } #sidebar::-webkit-scrollbar-thumb:hover { background: var(--color-text-tertiary); } /* Firefox */ #sidebar { scrollbar-width: thin; scrollbar-color: var(--color-bg-hover) transparent; }

Browser Compatibility Notes

  • Chrome/Edge/Safari: Full support via ::-webkit-scrollbar pseudo-elements
  • Firefox: Uses scrollbar-width and scrollbar-color properties. No hover state support.
  • Known limitation: Firefox does not support hover state changes on scrollbar thumbs
